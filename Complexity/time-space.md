## Complexity 
1. Time Complexity: O(n)
2. Space Complexity: O(1)

1. Time Complexity 
    - 3 Different Notations 
        - Big O Notation: O(n) 
            - Worst Case Scenario 
        - Big Omega Notation: Ω(n) 
            - Best Case Scenario 
        - Big Theta Notation: Θ(n) 
            - Average Case Scenario
    
    - we although interseted in worst case scenario by this we can see the how the algorithm performs in best and average case scenario as well

2. Space Complexity
    - Space Complexity: O(1) 
        - Constant Space Complexity 
        - The algorithm uses a fixed amount of space regardless of the input size. 
        - Example: A function that uses a constant number of variables to perform its operations, such as a simple loop that iterates through an array without using any additional data structures.
